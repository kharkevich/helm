Get Started

To get started with our Helm charts, visit our GitHub repository.

The charts can be added using the following command:

helm repo add technicaldomain https://technicaldomain.github.io/helm

OCI repository is also supported:

helm repo add technicaldomain oci://ghcr.io/technicaldomain/helm
